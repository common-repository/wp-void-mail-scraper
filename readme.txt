Scraps email from text, .pdf ,.txt etc files, website . Verify email using API , add subscriber to mailpoet.
== Description ==

WP Void mail scraper can scrap mail from texts , files and also websites. It will only crawl the given URL website. You can verify emails using verifier and also will be able to add email lists to mailpoet subscriber list ! 


== Installation ==

1. Upload the plugin folder after extracting it to the `/wp-content/plugins/(the folder of the extracted plugin)` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to the WordPress dashboard of your site you will see a menu named WPVMS mail scraper
